Auth — /api/auth
POST /api/auth/register Create account ▶
Request body
{
  "username": "string",
  "email": "string",
  "password": "string"
}
Response 201
{
  "token": "jwt_token_string",
  "userId": "uuid",
  "username": "string"
}
POST /api/auth/login Sign in ▶
Request body
{
  "email": "string",
  "password": "string"
}
Response 200
{
  "token": "jwt_token_string",
  "userId": "uuid",
  "username": "string"
}
Generate — /api/study JWT required
POST /api/study/generate Generate from input ▶
Request body
{
  "text": "string",
  "type": "SUMMARY" | "QUIZ" | "FLASHCARDS",
  "subject": "string (optional)"
}
Response 200 — SUMMARY
{
  "sessionId": "uuid",
  "type": "SUMMARY",
  "subject": "string",
  "summary": "string",
  "createdAt": "ISO timestamp"
}
Response 200 — QUIZ
{
  "sessionId": "uuid",
  "type": "QUIZ",
  "subject": "string",
  "questions": [
    {
      "question": "string",
      "choices": ["A","B","C","D"],
      "correctIndex": 0,
      "explanation": "string"
    }
  ],
  "createdAt": "ISO timestamp"
}
Response 200 — FLASHCARDS
{
  "sessionId": "uuid",
  "type": "FLASHCARDS",
  "subject": "string",
  "flashcards": [
    {
      "front": "string",
      "back": "string"
    }
  ],
  "createdAt": "ISO timestamp"
}
OCR happens on-device. Android always sends plain text to this endpoint regardless of the original input type (image, PDF, or typed text).
History — /api/history JWT required
GET /api/history Get all sessions ▶
Response 200
[
  {
    "sessionId": "uuid",
    "type": "QUIZ" | "SUMMARY" | "FLASHCARDS",
    "subject": "string",
    "createdAt": "ISO timestamp"
  }
]
Returns a lightweight list — no full content. Used to populate the history screen.
GET /api/history/{sessionId} Get one session ▶
Response 200
Full session object
(same shape as /generate response)
Called when user taps a history item to reopen it.
DELETE /api/history/{sessionId} Delete a session ▶
Response 204
No content
